# 对数线性模型

对数线性模型在 NLP 中的应用非常广泛，它的主要优点是能够利用丰富的特征集，从而相比前面提到的简单的评估方法能更好的表示数据。本章将引出对数线性模型，讨论其定义和参数估计的方法。
