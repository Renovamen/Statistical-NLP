# 生成标注模型

# 定义

生成模型可以被用于标注问题。令 K\mathcal{K}K 为所有可能出现的标签的集合，且假设它是一个有限集。则有定义 2.1（生成标注模型，Generative Tagging Models）：

假设有限集 V\mathcal{V}V 是所有单词的集合，有限集 K\mathcal{K}K 是所有标签的集合。定义 S\mathcal{S}S 为所有序列对 ⟨x1…xn,y1…yn⟩\langle x_1 \dots x_n, y_1 \dots y_n \rangle⟨x1​…xn​,y1​…yn​⟩（n≥0,xi∈V,yi∈K,i=1…nn \geq 0, x_i \in \mathcal{V}, y_i \in \mathcal{K}, i = 1 \dots nn≥0,xi​∈V,yi​∈K,i=1…n） 的集合。则生成标注模型是一个函数 ppp，满足：

  1. 对任意 ⟨x1…xn,y1…yn⟩∈S\langle x_1 \dots x_n, y_1 \dots y_n \rangle \in \mathcal{S}⟨x1​…xn​,y1​…yn​⟩∈S，p(x1…xn,y1…yn)≥0p(x_1 \dots x_n, y_1 \dots y_n) \geq 0p(x1​…xn​,y1​…yn​)≥0

  2. ∑⟨x1…xn,y1…yn⟩∈Sp(x1…xn,y1…yn)=1\sum_{\langle x_1 \dots x_n, y_1 \dots y_n \rangle \in \mathcal{S}} p(x_1 \dots x_n, y_1 \dots y_n) = 1∑⟨x1​…xn​,y1​…yn​⟩∈S​p(x1​…xn​,y1​…yn​)=1

也就是说 p(x1…xn,y1…yn)p(x_1 \dots x_n, y_1 \dots y_n)p(x1​…xn​,y1​…yn​) 是序列对的概率分布。

从 x1…xnx_1 \dots x_nx1​…xn​ 到 y1…yny_1 \dots y_ny1​…yn​ 的映射的定义为：

f(x1…xn)=arg⁡max⁡y1…ynp(x1…xn,y1…yn)f(x_1 \dots x_n) = \arg \max_{y_1 \dots y_n} p(x_1 \dots x_n, y_1 \dots y_n) f(x1​…xn​)=argy1​…yn​max​p(x1​…xn​,y1​…yn​)

也就是说模型的输出结果是概率最高的标注序列。

# 存在的问题

现在有三个重要的问题：

  • 如何定义 p(x1…xn,y1…yn)p(x_1 \dots x_n, y_1 \dots y_n)p(x1​…xn​,y1​…yn​)
  • 如何从训练集中估计参数
  • 怎样高效的找出 arg⁡max⁡y1…ynp(x1…xn,y1…yn)\arg \max_{y_1 \dots y_n} p(x_1 \dots x_n, y_1 \dots y_n)argmaxy1​…yn​​p(x1​…xn​,y1​…yn​)
