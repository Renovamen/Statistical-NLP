# 标注问题和隐马尔科夫模型

标注问题（tagging problem）（或序列标注问题（sequence labeling problem））是将源序列 x1...xnx_1...x_nx1​...xn​（标注模型的输入）映射到标记序列 y1...yny_1...y_ny1​...yn​（标注模型的输出）的问题。

假设我们有训练集 (x(i),y(i)),i=1...m(x^{(i)},y^{(i)}),i = 1...m(x(i),y(i)),i=1...m，其中 x(i)x^{(i)}x(i) 为句子 x1(i)...xni(i)x_1^{(i)}...x_{n_i}^{(i)}x1(i)​...xni​(i)​， y(i)y^{(i)}y(i) 为标签序列 y1(i)...yni(i)y_1^{(i)}...y_{n_i}^{(i)}y1(i)​...yni​(i)​。即 xj(i)x^{(i)}_jxj(i)​ 是第 iii 个训练句子的第 jjj 个单词，y(i)y^{(i)}y(i) 是它的标签。那么标注问题就是从该训练集中学习出一个能把这些句子映射到其标签序列的函数。
